/trending_searches/<region> - возвращает список самых популярных запросов в указанной стране (например RU - Россия) в виде json
/interest_over_time/<string:region>/<string:keyword> - возвращает словарь {дата:интерес (от 0 до 100)} запроса keyword в указанной стране (например RU - Россия) в виде json
