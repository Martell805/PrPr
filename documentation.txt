/trending_searches/<region> - возвращает список самых популярных запросов в указанной стране (например RU - Россия, WR - мир) в виде json
/interest_over_time/<region>/<keyword> - возвращает список словарей {"date": <дата>, "popularity": <популярность>} запроса keyword в указанной стране (например RU - Россия, WR - мир) в виде json
/info/<keyword> - возвращает словарь вида с ключём "page_exists": <существование страницы> и, если он True, "short": <короткая информация о запросе> и "full": <полная информация о запросе>
/full_info/<region>/<keyword> - возвращает словарь, аналогичный info, но с "graph": <список точек, как в interest_over_time>
