/trending_searches/<region> - возвращает список самых популярных запросов в указанной стране (например RU - Россия, WR - мир)


/interest_over_time/<region>/<keyword> - возвращает список словарей
{
    "date": <дата>,
    "popularity": <популярность>
}
запроса keyword в указанной стране (например RU - Россия, WR - мир)


/related_searches/<region>/<keyword> - возвращает список словарей
{
    "keyword": <запрос>,
    "type": <тип запроса (что это одним словом)>,
    "relation": <связанность>
}
запроса keyword в указанной стране (например RU - Россия, WR - мир)


/info/<keyword> - возвращает словарь вида
{
    "page_exists": <существование страницы>,
    "short": <короткая информация о запросе>,
    "full": <полная информация о запросе>
}
"short" и "full" есть только если "page exists" - true


/full_info/<region>/<keyword> - возвращает словарь вида
{
    "page_exists": <существование страницы>,
    "short": <короткая информация о запросе>,
    "full": <полная информация о запросе>,
    "graph": <список точек:
        {
            "date": <дата>,
            "popularity": <популярность>
        }>
}
"short" и "full" есть только если "page exists" - true
